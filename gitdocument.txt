We need to create three seperate branch for Dev, UAT, Prod

Step 1 : Create three branch in a single repository.
         command : git branch dev
                   git branch uat
                   git branch prod
Step 2 : To display list of branch : 
          command : git branch 
          Note : * symbol represent the current working branch.

Step 3 : Push all the branches on github to simulate the same code.
         command : git checkout prod
                   git add .
                   git commit -m "Inital commit"

Step 4 : command ; git push origin dev
                   note : dev is optional
